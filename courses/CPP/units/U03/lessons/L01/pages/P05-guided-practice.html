Guided Practice

Think about one school or career-related goal you want to improve this semester.

Write responses to all of the following:

  • State your goal in one sentence.
  • Explain how the goal is specific.
  • Explain how the goal is measurable.
  • Explain how the goal is achievable and relevant.
  • Add a time frame to make the goal time-bound.

Use complete sentences and make sure each part of SMART is clear.
