💡Worked Example

Scenario: Jordan includes a project summary in a portfolio, but the summary has no title and does not explain what skill the project shows.

Feedback Jordan receives: "Add a clear title. Explain what the project demonstrates. Make the formatting easier to read."

Jordan revises by:

  • adding a title
  • writing a short explanation of the skill shown
  • improving spacing and labels

Result: The revised artifact is easier to understand and more useful to a reviewer.

✅Why this works: Jordan used feedback to make the item clearer, stronger, and more professional.
