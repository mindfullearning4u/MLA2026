💡Worked Example

💡Read the sample scenario and observe how secure login practices protect accounts.

Scenario: A student uses the same password for school, gaming, and social media. The password is "soccer123." One of the gaming sites gets hacked, and the password leaks online.

Why this is unsafe:

  • The password is short and easy to guess.
  • Reusing passwords means one leak exposes all accounts.
  • Hackers can log in without needing extra verification.

Safer account security choices:

  • Create a long, unique password for each account.
  • Turn on MFA so a code is required to log in.
  • Update old or weak passwords immediately.

✅Why this works:

  • It prevents hackers from accessing multiple accounts.
  • It adds extra protection even if a password leaks.
  • It models responsible digital security habits.
