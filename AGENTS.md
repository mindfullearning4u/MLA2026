# MLA2026 Curriculum Build Rules (LOCKED STANDARD)

## NON-NEGOTIABLE RULES

1. Lesson content is LOCKED.
   - Do NOT modify, rewrite, expand, or simplify any instructional content.
   - Do NOT change examples, scenarios, or wording.

2. Allowed changes:
   - File structure creation
   - JSON field updates (IDs, titles, order)
   - HTML formatting for LMS compatibility
   - Standard wording change ONLY:
     - "Upload" → "Submit"

3. Not allowed under any circumstance:
   - Changing lesson meaning
   - Rewriting worked examples
   - Adding new instructional content
   - Removing any provided content

4. If improvement is identified:
   - DO NOT implement
   - Present as suggestion ONLY

5. Unit 1 is the GOLD STANDARD.
   - All future units must match its structure exactly.

6. Every lesson MUST include:
   - lesson.json
   - quiz.json
   - 7 pages (P01–P07)

7. Naming conventions are strict and must not change.

## EXECUTION MODE

- Operate in "structure-only mode"
- Treat all lesson content as immutable source material
